# I Don't Need HWP Table

> **한글 표, 너의 시대는 끝났다.**

---

## 왜 이게 필요한가

2026년이다. 전 세계가 마크다운 쓰고, JSON 쓰고, API 날리는 시대에  
대한민국 공공기관·기업·학교는 여전히 `.hwp` 파일 안에 표를 구겨넣는다.

그 표를 AI한테 던지면?  
**AI는 눈을 감는다.**  
복사해서 붙여넣으면? **셀이 다 합쳐진다.**  
캡처해서 올리면? **OCR이 울부짖는다.**

이 프로젝트는 그 분노에서 태어났다.

---

## 뭘 하는 도구인가

`.hwpx` 파일을 열고, 표를 꺼내고, 쓸 수 있는 형식으로 바꿔준다.

| 출력 형식 | 용도 |
|---|---|
| **HTML** | 웹에 그냥 붙여넣기 |
| **마크다운** | GitHub, Notion, 어디든 |
| **AI 프롬프트** | GPT, Claude한테 표 내용 먹이기 |

빌드 없음. 서버 없음. 그냥 `index.html` 열면 끝.

---

## 사용법

1. `index.html` 브라우저에서 열기
2. `.hwpx` 파일 드래그 앤 드롭
3. 왼쪽에서 표 선택
4. 원하는 형식 탭 클릭 → 복사

---

## 지원 형식

- ✅ `.hwpx` (한글 2010 이상, XML 기반)
- ❌ `.hwp` (구형 바이너리 — 한글에서 "다른 이름으로 저장 → .hwpx"로 변환 후 사용)

---

## 기술 스택

- Vanilla JS + HTML + CSS
- [JSZip](https://stuk.github.io/jszip/) — `.hwpx`(ZIP) 압축 해제용

그게 전부다. 프레임워크가 필요 없다.

---

## 마치며

한글 표가 나쁜 게 아니다.  
한글 표를 **이메일로 보내고, 인쇄해서 결재받고, 스캔해서 다시 보내는** 그 문화가 문제다.

언젠가 이 도구가 필요 없는 날이 오길 바라며.  
그 전까지는 쓴다.
